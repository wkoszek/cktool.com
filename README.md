# cktool-com

Website source for [CK Tool](https://cktool.com), a precision CNC machining and robotic automation shop.

## Structure

```
index.html      # Main page
privacy.html    # Privacy policy
terms.html      # Terms of service
img/            # Images and assets
robots.txt      # SEO robots file
sitemap.xml     # XML sitemap
```

## Development

Open `index.html` in a browser. Static HTML — no build step required.

## Deployment

Deploy via GitHub Pages or any static host. Update `CNAME` for custom domain.
